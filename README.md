# Stocks App

A comprehensive real-time stock market dashboard application built with Next.js. This application provides users with up-to-date market data, news, and analytical tools.

![Daily News Summary](Daily%20News%20Summary.png)

## Features

- **Real-time Market Overview**: Track global market indices and performance at a glance.
- **Stock Heatmap**: Visual representation of stock market performance across different sectors.
- **Top Stories & News Timeline**: Stay informed with the latest financial news and market-moving stories.
- **Market Quotes**: Get detailed quotes and data for specific stocks.
- **Authentication**: Secure user authentication system for personalized experiences.

## Authentication

The application features a robust authentication system allowing users to sign in and sign up to access personalized features.

![Stocks App Auth](Stocks%20App%20Auth.png)

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Authentication**: Better Auth
- **Database**: MongoDB (via Mongoose)
- **Inngest**: Inngest
- **UI Components**: Radix UI, Lucide React
- **Charts/Widgets**: TradingView Widgets

## Getting Started

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

5. Inngest Production link: [https://app.inngest.com/env/production/apps](Inngest)
